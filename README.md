# JS2CS

A NodeJS to C# Websocket Wrapper

## Description

Creates a Heartbeat between NodeJS and C#.

## Getting Started

### Dependencies

* Visual Studio

### Executing

* Clone the repository
* Run the Node
* Use the Example provided or in your own project.
* Sends Status Code and Disconnection Code
